# Security Policy

## Reporting a Vulnerability

Please do not file public issues for security-sensitive reports.

Preferred: Report privately via GitHub Security Advisories:
https://github.com/EmekaOkaforTech/custos-core/security/advisories

If Advisories are unavailable for any reason, contact:
- Email: contact@okafor.tech
- GitHub: @EmekaOkaforTech

We will acknowledge receipt within a reasonable timeframe and coordinate a fix and disclosure plan.
